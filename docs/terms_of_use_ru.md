# Условия использования платформы TmWork

## 1. Общие положения
Пользуясь сайтом TmWork, вы соглашаетесь с данными условиями. Платформа является посредником между Заказчиками и Исполнителями.

## 2. Регистрация и роли
- Пользователь обязан предоставлять достоверную информацию.
- Один аккаунт может использоваться как для роли Клиента, так и Фрилансера.

## 3. Система Безопасной Сделки (Escrow)
- При принятии отклика средства Заказчика блокируются в системе.
- Выплата происходит после подтверждения выполнения работ или по решению Арбитража.
- Комиссия платформы указывается при совершении сделки.

## 4. Запрещенные действия
- Обмен контактами до заключения сделки (в целях обхода комиссии).
- Публикация незаконного контента.
- Мошенничество и использование уязвимостей системы.

## 5. Разрешение споров
Все споры решаются через внутреннюю систему Арбитража. Решение Администратора является окончательным.
