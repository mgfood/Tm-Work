# 🚀 Полная инструкция по запуску в интернет (Localtunnel)

Чтобы твой друг мог зайти на сайт, выполни эти шаги по порядку. Тебе понадобится **4 отдельных окна терминала**.

### 1️⃣ Терминал: Запуск Бэкенда (Django)
```powershell
cd f:\TmWork\backend
# Активируй окружение, если нужно
python manage.py runserver
```

### 2️⃣ Терминал: Публикация Бэкенда
Здесь мы получаем "внешний" адрес для твоего API:
```powershell
lt --port 8000
```
> **СКОПИРУЙ** полученную ссылку (например, `https://smart-cats-jump.loca.lt`)

### 3️⃣ Обновление настроек
Открой файл `f:\TmWork\frontend\.env` и вставь туда эту ссылку:
```env
VITE_API_URL=https://smart-cats-jump.loca.lt
```

### 4️⃣ Терминал: Запуск Фронтенда (React)
```powershell
cd f:\TmWork\frontend
npm run dev
```

### 5️⃣ Терминал: Публикация Сайта
Теперь делаем сам сайт доступным для друга:
```powershell
lt --port 3000
```
> **Эту ссылку** (например, `https://cool-site.loca.lt`) отправляй другу!

---

### ❓ Что если ссылка не открывается?
1.  **IP Verification**: Localtunnel при первом открытии просит "Endpoint Password". Это просто твой внешний IP. Его можно узнать, вбив в Google "my ip" или на сайте [whatsmyip.org](https://www.whatsmyip.org/). Скопируй его и вставь на странице Localtunnel.
2.  **Fallback**: Если в `.env` файле будет пусто или ошибка в ссылке, фронтенд автоматически попробует соединиться с `127.0.0.1:8000` (локально).
3.  **Перезапуск**: Если ты закроешь терминал с `lt`, ссылка изменится. Нужно будет снова обновить `.env`.
