# 🚀 Инструкция по запуску в интернет (Localtunnel)

Localtunnel — это отличный бесплатный инструмент, который позволяет сделать ваш локальный сайт доступным по ссылке.

## 🛠 Подготовка
Установите localtunnel глобально:
```powershell
npm install -g localtunnel
```

## 👣 Пошаговый запуск (Нужно 4 терминала)

### 1. Запуск Бэкенда (Порт 8000)
В первом терминале запустите Django:
```powershell
cd backend
python manage.py runserver
```

### 2. Запуск тоннеля для Бэкенда
Во втором терминале создайте публичную ссылку для API:
```powershell
lt --port 8000
```
**Важно**: Скопируйте полученную ссылку (например, `https://smart-cats-jump.loca.lt`).

### 3. Настройка Фронтенда
Откройте файл `frontend/.env` и замените `VITE_API_URL` на вашу новую ссылку:
```env
VITE_API_URL=https://smart-cats-jump.loca.lt
```

### 4. Запуск Фронтенда (Порт 3000)
В третьем терминале запустите Vite:
```powershell
cd frontend
npm run dev
```

### 5. Запуск тоннеля для Фронтенда
В четвертом терминале создайте публичную ссылку для сайта:
```powershell
lt --port 3000
```
**Эту ссылку** (например, `https://happy-walls-dance.loca.lt`) можно отправлять другу!

---

## ⚠️ Важные моменты:
- **IP-проверка**: При первом открытии ссылки Localtunnel может попросить ввести ваш внешний IP-адрес. Его можно узнать на сайте [whatsmyip.org](https://www.whatsmyip.org/), просто скопируйте его и вставьте в поле на странице localtunnel.
- **Постоянство**: Каждый раз при перезапуске `lt`, ссылка будет меняться. Не забывайте обновлять `VITE_API_URL` в `.env` файле фронтенда.
- **HTTPS**: Localtunnel автоматически дает HTTPS ссылки, что важно для безопасности.
